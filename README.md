# Crop Disease Detection and Farm Advisory System

This project detects crop diseases from leaf images and provides farm advisory, history, dashboard, irrigation calculator, PDF report, and farmer login system.

Frontend and Flask backend are developed in VS Code.
Model training backend is available in Google Colab.
